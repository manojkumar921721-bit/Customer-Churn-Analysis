# Customer Churn Analysis

## Project Overview

This project analyzes customer churn behavior in a telecom company using SQL, Python, and Power BI. The objective is to identify factors influencing customer attrition and provide actionable insights to improve customer retention.

## Key Insights

* Month-to-month customers exhibit the highest churn rate.
* Electronic check users show higher churn behavior.
* Customers with higher monthly charges are more likely to churn.
* Longer customer tenure improves retention.
* Contract duration significantly impacts churn.

---

## Business Recommendations

* Encourage customers to move to long-term contracts.
* Focus retention campaigns on high-risk customers.
* Improve engagement among short-tenure customers.
* Offer incentives for customers using high-risk payment methods.
